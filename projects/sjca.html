In March of 2025, I was commissioned by San Jose Christian Assembly to revamp their website.

My local church, San Jose Christian Assembly, approached me to modernize their web presence. The previous website was outdated and lacked interactive features that could engage the congregation. Using modern web technologies like TypeScript and Next.js, I developed a dynamic and responsive website that serves both the congregation and administrative needs.

The website features several key interactive elements, including a readable Bible with verse highlighting and note-taking capabilities, an interactive timeline showcasing the church's history, and animated page transitions that create a polished user experience. The site is built with a mobile-first approach, ensuring accessibility across all devices.

One of the major improvements is the integration of a Supabase database, allowing administrative users to easily update content without touching code. This includes managing event announcements, sermon archives, and ministry updates. The admin dashboard provides a user-friendly interface for content management, while maintaining security through role-based access control.

The website is deployed on Vercel, ensuring fast load times and reliable performance. The deployment pipeline is automated, making updates seamless and immediate. The tech stack choice of Next.js enables server-side rendering, improving both performance and SEO rankings.

Additional features include:

  • Real-time prayer request system
  • Integrated calendar for church events
  • Sermon archive with search functionality
  • Ministry team profiles and contact information
  • Automated email notifications for important updates
