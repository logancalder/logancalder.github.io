Este proyecto HTML fue mi antiguo sitio web que desde entonces ha sido reemplazado por el que estás ahora.

Lo que ves aquí es lo que solía ser este sitio web, desde 2020-2024. Este fue mi primer proyecto HTML, del cual aprendí los conceptos básicos del diseño web frontend y la interfaz de usuario. Aunque no es extremadamente bonito, creé este sitio web completamente por mi cuenta, sin la ayuda o contribución de nadie más. Solo yo, algo de YouTube y 10 horas de mi tiempo. A partir de ahí, el sitio web creció para presentar nuevos proyectos y actualizar mi experiencia laboral.

Quería algo que destacara, algo que me hiciera único. En un mundo de infinitos ingenieros de software y desarrolladores, un currículum simplemente no te diferencia del resto de la multitud. Sabiendo esto, quería una forma llamativa de mostrar mi trabajo pasado, habilidades y otra información sobre mí.

Me inspiré en iuri.is (puedes encontrar su trabajo enlazado aquí). Estudiando su escritura HTML y CSS (usando la función inspeccionar en un navegador), aprendí técnicas de formato y estilo que en ese momento eran desconocidas para mí. Insertando estas en mi propio trabajo, el sitio web comenzó a tomar forma lentamente.

También aprendí a escribir HTML y CSS de W3Schools, un gran recurso, y StackOverflow. ChatGPT aún no existía en el momento en que creé este sitio, por lo que esa herramienta no se utilizó para este proyecto.


Si deseas ver el proyecto tú mismo, he subido el código a un repositorio de GitHub enlazado aquí.


Para los nodos interactivos en la pantalla de inicio, utilicé una biblioteca JS de código abierto que no puedo encontrar de nuevo :(
