Cuestionario de San Valentín 2024

Este proyecto fue mi pequeña forma divertida de pedirle a mi novia que sea mi Valentín este año.

Había visto muchas publicaciones diferentes de personas pidiendo a sus seres queridos que sean sus Valentines, ya sea a través de una tarjeta o flores, pero quería hacer algo diferente. Algo original. Y así, decidí desarrollar un sitio web interactivo que le preguntaría infinitamente hasta que dijera que sí (¡porque no puede decir que no)!

El sitio web consiste en un corazón animado que late, animaciones de texto desvanecidas y múltiples indicaciones de texto que cambian en cada clic de botón. Las indicaciones se ciclarán infinitamente hasta que se haga clic en el botón "sí", revelando luego una pantalla de celebración con un GIF lindo.

Este proyecto me obligó a obtener una comprensión mucho más fuerte de JavaScript de la que ya tenía, y también decidí escribir HTML y CSS ordenados y limpios (sin código espagueti aquí). Estoy bastante contento con cómo resultó, ya que creé el proyecto completamente por mi cuenta, y de este proyecto desarrollé una nueva pasión por el desarrollo frontend y el diseño UI/UX.


Si desea ver el proyecto usted mismo, he subido el código a un repositorio de GitHub vinculado aquí.
