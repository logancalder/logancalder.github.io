Sitio Web de CMTM

Me pidieron que desarrollara un sitio web para la Misión de Entrenamiento del Ministerio Cristiano.

En el otoño de 2023, la Misión de Entrenamiento del Ministerio Cristiano (CMTM) me pidió que creara un sitio web para ellos que mostrara su trabajo, objetivos y sirviera como un centro general para que las personas se pusieran en contacto con ellos. Anteriormente, no tenían sitio web y carecían de una forma de mostrar su misión o reclutar más misioneros.

Dado este objetivo, creé un sitio web interactivo que presentaba una página de inicio informativa y gráficamente saturada, una página sobre en profundidad, así como una galería de fotos y una colección de música. El sitio web también cuenta con una opción de idioma para cambiar entre mandarín tradicional e inglés, ya que la mayoría de los misioneros que trabajan para el ministerio, así como el público objetivo, son hablantes fluidos o inmigrantes de un país de habla. Sabiendo que el público objetivo estaba en las generaciones mayores, el sitio web fue diseñado para ser fácil de navegar y amigable para el usuario.

Para este proyecto estudié otros sitios web de ministerios populares para evaluar cuál sería el formato más común y efectivo. Construir este sitio fue relativamente fácil con pocos o ningún obstáculo, pero tuve que idear una forma de integrar dos idiomas. Todos los métodos que había visto en línea parecían excesivos y demasiado trabajo, además de hacer que el código HTML se viera desordenado con dos copias del mismo dispuestas en el mismo documento. Dado esto, decidí integrar los dos idiomas teniendo un botón que navegara a otro directorio que almacenara los archivos escritos en mandarín.


Si desea ver el proyecto usted mismo, he subido el código a un repositorio de GitHub vinculado aquí.

El sitio web aún se está publicando, ¡pero lo vincularé tan pronto como esté en línea!
